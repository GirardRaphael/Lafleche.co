# Lafleche.co Website

Welcome to the Lafleche.co website repository! This repository contains the HTML website for your business.

## Structure

```
Lafleche.co/
├── index.html          # Main landing page
├── assets/             # Static assets directory
│   ├── css/           # Stylesheets
│   │   └── style.css  # Main stylesheet
│   ├── js/            # JavaScript files
│   │   └── main.js    # Main JavaScript file
│   └── images/        # Image assets
├── .gitignore         # Git ignore file

```

## Getting Started

### Adding Your HTML Files

You can now transfer your existing HTML website files into this repository:

1. **Main Pages**: Place your HTML pages in the root directory (alongside index.html)
2. **Styles**: Add your CSS files to `assets/css/`
3. **Scripts**: Add your JavaScript files to `assets/js/`
4. **Images**: Add your images to `assets/images/`

### Customizing the Template

The included `index.html` is a starter template. You can:
- Replace it entirely with your existing HTML
- Use it as a reference for structure
- Modify it to match your business needs

### Viewing the Website Locally

To view your website locally:

1. Simply open `index.html` in a web browser
2. Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```
3. Visit `http://localhost:8000` in your browser

## Deployment

### GitHub Pages

This website is ready to be deployed on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select the branch (main or copilot/transfer-html-website)
4. Select the root folder
5. Click "Save"

Your website will be available at: `https://girarddraphael.github.io/Lafleche.co/`

### Other Hosting Options

You can also deploy to:
- **Netlify**: Drag and drop the folder or connect your GitHub repository
- **Vercel**: Connect your GitHub repository
- **Traditional hosting**: Upload files via FTP
